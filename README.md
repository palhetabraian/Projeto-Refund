# 💸 Sistema de Reembolso

Aplicação web simples para adicionar e visualizar despesas para reembolso.

## 🚀 Funcionalidades
- Adicionar despesas (nome, categoria e valor)
- Listar despesas
- Exibir total acumulado

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript
